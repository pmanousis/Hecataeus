All Modes:

  • Right-click on an empty area for Create Vertex
  • Right-click on a Vertex for Delete Vertex
  • Right-click on a Vertex for Delete Vertices (if more than one vertices are selected)
  • Right-click on a Vertex for Add Policy (if more than one vertices are selected)
  • Right-click on a Vertex for Edit Vertex
  • Right-click on a Vertex for Add Edge menus (if there are selected Vertices)
  • Right-click on a Vertex for Events available choices
  • Right-click on a Vertex for Policies available choices
  • Right-click on an Edge for Delete Edge
  • Right-click on an Edge for Edit Edge
  • Click on an empty area for removing highlight
  • Mousewheel for scaling

Picking Mode:

  • Mouse1 on a Vertex selects the vertex
  • Mouse1 elsewhere unselects all Vertices
  • Mouse1Shift on a Vertex adds/removes Vertex selection
  • Mouse1drag on a Vertex moves all selected Vertices
  • Mouse1drag elsewhere selects Vertices in a region
  • Mouse1Shiftdrag adds selection of Vertices in a new region

Moving Mode:

  • Mouse1drag pans the graph
  • Mouse1Shiftdrag rotates the graph
  • Mouse1CTRL(or Command)drag shears the graph
